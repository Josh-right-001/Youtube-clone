# Youtube-clone
C'est un challenge qui consiste à créer un clone de YouTube en s'inspirant d'un modèle de tutoriel vidéo en utilisant un framework | ReactJs et les API YouTube. Ce projet met en œuvre des requêtes API externes pour récupérer et afficher des données vidéo de YouTube.
